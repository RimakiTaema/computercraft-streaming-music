# 3.0.0_vibe - Major UI Overhaul

## Visual
- New cyan accent color theme
- Active tab highlight with underline indicator
- Animated loading dots
- Music note and playlist icons on results
- Flash feedback on button clicks
- Status bar with keybind hints

## Killer Feature
- Real-time audio visualizer on Now Playing screen
- Bars react to the DFPWM audio buffer as it plays
- Purple peaks, cyan body, smooth decay

## New Controls
- Keyboard navigation: Left/Right for tabs
- Space to play/pause, N to skip, Q to go back
- +/- to adjust volume, S to open search
- Mouse wheel scroll on all lists

## New Features
- Shuffle queue button
- Clear queue button
- Click queue items to remove them
- Scrollable search results, queue, and changelogs
- Scroll indicators when content overflows
- Auto-fetch changelogs on tab open
- Play Now auto-switches to Now Playing tab

## Backend
- Switched from Firebase Cloud Functions to standalone server
- Uses yt-dlp + ffmpeg pipeline for audio
- No more RapidAPI dependency for downloads
- dotenv for environment config
- yt-dlp fallback: if RapidAPI quota runs out or key is missing, search/metadata automatically falls back to yt-dlp
- Works with zero API keys configured (fully yt-dlp powered)
